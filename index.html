Course Overview

SwamiG Institute • 501(c)(3)

This curriculum advances the student beyond basic yes/no divination. Four-Cowrie work establishes the foundation language and discipline; Orisha Pele extends that foundation into deeper interpretive movement and applied reading.

Outcome: a student who can cast, interpret, and communicate results with structure, clarity, and competency — culminating in certification completion.

Prerequisites

Readiness checklist
  1. Four-Cowrie Shell Divination experience
  2. 30 days minimum practice
  3. Dedicated casting surface
  4. Ability to attend weekly Zoom
  5. Access to Orisha Pele implements
  6. Commitment to certification

Four-Cowrie Shell Framework

Foundation language • 5 possibilities

The course uses the Four-Cowrie naming framework as the baseline. These states are the structural “anchor” for Orisha Pele expansions.

Open Name Notes
4 Ogbe Core reference state (foundation for Week 1)
3 Etawa Core reference state (foundation for Week 1)
2 Ejife Core reference state (foundation for Week 2)
1 Okana Core reference state (foundation for Week 3)
0 Oyeku Core reference state (foundation for Week 4–5 integration)
Key idea: Four Cowries establish structure. Orisha Pele reveals movement, nuance, and layered outcomes.

Weekly Modules (5 Weeks)

Live Zoom + documented practice

Week 1 Foundation • Ogbe + Etawa

Orientation and transition from “yes/no” into expanded interpretive language.

  • Ogbe (core anchor)
  • Etawa (core anchor)
  • Etawa expansions: Obita, Akita

Week 2 Expansion • Ejife

Stability themes, pattern recognition, and building interpretive confidence.

  • Ejife (core anchor)
  • Ejife expansions: Ejire, Akoran, Eros

Week 3 Application • Okana

Material/health axes and practical reading structure.

  • Okana (core anchor)
  • Okana expansions: Aje, Ilera

Week 4 Synthesis • Oyeku

Deeper interpretation: transformation, closure, and continuity.

  • Oyeku (core anchor)
  • Combining prior weeks into coherent outcomes
  • Refining language: clarity + responsibility

Week 5 Integration • Full Flow

Complete casting flow, communication practice, and readiness review.

  • Full-cycle casting practice (start → interpret → conclude)
  • Applied readings + instructor feedback
  • Preparation for certification competency
Suggestion: Keep a practice log each week (date, question, cast result, meaning, action). This supports evaluation and strengthens your interpretive discipline.

Certification Completion

Competency + Ceremony

Successful completion includes a Certification Competency Ceremony. Students demonstrate practical casting, interpretation, and responsible communication.

  • Demonstrate consistent casting process
  • Accurately identify core states + expansions
  • Present a structured reading (clear + grounded)
  • Show ethical boundaries and appropriate next steps
